# Sahyog_project


## Problem Statement
The NGO ecosystem is fragmented and lacks a unified digital platform, leading to inefficient coordination, limited beneficiary accessibility, and low transparency in social impact operations.

## Solution
SAHYOG is a centralized platform connecting NGOs, volunteers, donors, and beneficiaries to streamline social impact coordination and improve accessibility, transparency, and operational efficiency.

## Features
- NGO Registration and Dashboard
- Beneficiary Request Management
- Volunteer Coordination
- Donation Tracking
- NGO Search and Discovery
- Alerts and Announcements

## Tech Stack
- HTML
- CSS
- JavaScript
- Google Cloud
- Gemini API 

## Future Scope
- AI-Based NGO Recommendation
- Geolocation NGO Matching
- Emergency Request Prioritization
- Mobile Application
